# weather-app

## requirements

Vite requires Node.js version 18+. 20+. However, some templates require a higher
Node.js version to work [link to install nodejs](https://nodejs.org/en)

Install recommended extensions showed as pop up in vscode!

## how to run project:

```bash
    npm i
    npm run dev
```
